## Common Questions (FAQ)

**Q: Can I modify the template design/colors?**  
A: Yes! Customize colors, fonts, layouts as you wish. Just meet all requirements.

**Q: The template has 4 pages. Can I add more?**  
A: Yes, but you MUST have these 4 required pages. Extra pages (like a game page for extra credit) are welcome.

**Q: Do I have to use all the example sections?**  
A: Yes for required sections (marked in comments). Optional sections can be removed.

**Q: Can I use different CSS frameworks (Bootstrap, Tailwind)?**  
A: Yes, but the provided CSS already meets all requirements. Not necessary to change.

**Q: What if I don't have much work experience for my resume?**  
A: Include part-time jobs, volunteer work, school projects, or leadership roles. Check comments in resume.html for guidance.

**Q: How long should my team video be?**  
A: **At least 30 seconds.** Longer is okay, but keep it concise and purposeful!

**Q: Can each team member use a different video?**  
A: NO! All 5 team members must use the SAME team video (30+ seconds).

**Q: What if I can't find company logos?**  
A: Check the company's website footer or "Press Kit" / "Media" / "About" pages. Most companies provide official logos for download.
